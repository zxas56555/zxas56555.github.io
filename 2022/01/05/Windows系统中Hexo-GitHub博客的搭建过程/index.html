Windows系统中Hexo+GitHub博客的搭建过程

2022-01-05 540 字

一.首先需要安装两个程序

  • node.js：http://nodejs.cn/download
  • Git：http://nodejs.cn/download/
  • 这里有一个可以用来确定软件/模块是否正确安装的方法，就是输入一下获取版本号的指令,比如 npm -v 或者 node -v等。*

二. 然后就是关于cnpm的配置

  • 大概就是因为npm贼慢贼卡，所以要配置一个叫做cnpm的东西，还要和淘宝链接，具体为啥是淘宝我也不知道，能用就行。
npm install -g cnpm --registry=http://registry.npm.taobao.org

三.安装Hexo

cnpm install hexo-cli -g

四.初始化一个博客

hexo init blog//在名为blog的文件夹中初始化hexo博客，如果blog文件夹存在且不为空，则会报错
hexo s//启动本地服务器，一般用于预览博客效果，打开浏览器输入localhost:4000即可查看，启动之后可以按Ctrl+C来终止

五.创建一篇新博客

  • 方法一：将一篇已经写好的 .md文件放到 blog\source\_post目录下。
  • 方法二：运行指令 hexo n ""，其中文件名也就是博客的标题写在引号中间，该指令会在 blog\source\_post目录下创建一个 .md文件，然后编辑这个文件就可以了。

六.静态文件的清理与生成

  • 每次对博客进行开动之后都要先清理再生成一遍静态文件，这个操作就好比是我们平常编辑一个文档之后要保存一样。

    hexo clean//清理静态文件
    hexo g//生成静态文件

七.将博客部署到GitHub上

  1. 注册一个GitHub账号

  2. 新建一个仓库，名称一定要是 你的用户名.github.io

  3. 回到cmd，安装插件 cnpm install --save hexo-deployer-git

  4. 编辑blog目录下的 _config.yml:

    1. 拉到文件最底下，找到这样一块，并按照下面的格式填写，注意冒号后面都有空格。
    deploy:
    type: 'git'
    repo: https://github.com/你的用户名/你的用户名.github.io.git
    branch: master

  5. 回到cmd，输入部署指令 hexo d然后按照提示输入GitHub的账号密码什么的就行了。
