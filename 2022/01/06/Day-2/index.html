JSON序列化与反序列化

  • 需加载模块json
  • 序列化即将Python里的数据结构转化为JSON字符，例如：
import json
data = [{'b' : 2, 'd': 4, 'a': 1, 'c': 3}]
json_format = json.dumps(data, sort_keys = True, indent = 4, separators(',', ': '))
print(json_format)

上述代码输出为：

[
    {
        "a": 1,
        "b": 2,
        "c": 3,
        "d": 4
    }
]
  • 反序列化即将上述过程逆过来。

Python爬虫

  1. 环境安装：
    1. BeautifulSoup库
    2. lxml解析器
  2. 使用request库向指定网站发送请求
    1. get方法：request.get(url, head)
      1. 返回的是一个response对象，其中
        1. response.status_code：http请求的返回装套，200表示链接成功，404表示失败。
        2. response.text：HTTP相应内容的字符串形式，即url对应的页面内容
        3. response.encoding：从HTTP header中猜测的响应内容编码方式
        4. response.apparent_encoding：从内容中非吸出的响应内容的编码方式（备选编码方式）
        5. response.contenthttp：响应内容的二进制形式
  3. 寻找需要的信息
    1. 将返回的response对象创建BeautifulSoup对象，
    2. Beautiful Soup将复杂HTML文档转换成一个复杂的树形结构，每个节点都是Python对象。
    3. 利用Beautifulsoup库的功能寻找需要的数据
    4. Python中BeautifulSoup库的用法
