Day 3 - 深度学习常用库

NumPy

NumPy是使用Python进行科学计算的基础包。它包含如下的内容：

  • 一个强大的N维数组对象。
  • 复杂的（广播）功能。
  • 用于集成C / C ++和Fortran代码的工具。
  • 有用的线性代数，傅里叶变换和随机数功能。

除了明显的科学用途外，NumPy还可以用作通用数据的高效多维容器。可以定义任意数据类型。这使NumPy能够无缝快速地与各种数据库集成。

NumPy是在BSD许可下获得许可的，允许重用而不受限制。

*摘自NumPy中文网

N维数组（ndarray）

构造数组

  • 使用ndarry构造函数构建新数组：ndarray(shape[, dtype, buffer, offset, ...])

  • 将列表或元组转化为数组：

    • 列表：array([[1,2,3], [4,5,6]])
    • 元组：array(((1,2,3), (4,5,6)))

  • 创建制定长度或者形状的全零数组：zeros(shape[, dtype, order])

  • 创建指定长度或者形状的全一数组：ones(shape[, dtype, order])

  • 创建制定长度或形状的数组，内容取决于内存的状态：empty()

  • 返回给定间隔内的均匀间隔的值：arange([start,] stop[, step,][, dtype])

  • 重新定义数组的形状：reshape([2,3])

数组的计算

  • 矩阵的基础运算
import numpy

arr1 = numpy.array([[1,2,3],[4,5,6]])
arr2 = numpy.ones([2,3],dtype=numpy.int64)

print(arr1 + arr2) # 相加
print(arr1 - arr2) # 相减
print(arr1 * arr2) # 对应元素相乘
print(arr1 / arr2) # 对应元素相除
print(arr1 ** 2) # 所有元素的2次方
  • 矩阵乘法：dot(a, b[, out]
  • 其他计算：
numpy.sum(arr, axis = 1) # axis=1,每一行求和 axie=0,每一列求和
numpy.max() # 最大值
numpy.argmax() # 最大值下标
# min同上
numpy.mean() # 均值

数组索引与切片

  • 基本切片语法是 i:j:k，其中 i 是起始索引，j 是停止索引，k 是步骤（k不为0）

Pandas库

  • 基于NumPy实现

Series

  • 是一个键值对组成的数组
  • 同一个数组中键的值允许相同
  • 可存储整数、浮点数、字符串、Python对象的类型的数据。

构造

  • Series([], index = [])：逗号前是值，逗号后是键
  • 可以用字典构造：
d = {'b': 1, 'a': 0, 'c': 2}
pd.Series(d)

运算

  • Series中最重要的一个功能是：它会在算术运算中自动对齐不同索引的数据，也就是说，相同键的值才会进行运算，例如：
obj1 = pd.Series({"Ohio": 35000, "Oregon": 16000, "Texas": 71000, "Utah": 5000})
print(obj1)
obj2 = pd.Series({"California": np.nan, "Ohio": 35000, "Oregon": 16000, "Texas": 71000})
print(obj2)
print(obj1 + obj2)
  • 输出结果为：
Ohio      35000
Oregon    16000
Texas     71000
Utah       5000
dtype: int64
California        NaN
Ohio          35000.0
Oregon        16000.0
Texas         71000.0
dtype: float64
California         NaN
Ohio           70000.0
Oregon         32000.0
Texas         142000.0
Utah               NaN
dtype: float64

DataFrame

构造

  • 可以使用字典进行构造：字典的键是dataframe第一行的值，字典的值应是一个列表，为dataframe对应列的每一行的值，例如：
data = {'state': ['Ohio', 'Ohio', 'Ohio', 'Nevada', 'Nevada'], 'year': [2000, 2001, 2002, 2001, 2002], 'pop': [1.5, 1.7, 3.6, 2.4, 2.9]}
frame = pd.DataFrame(data)
print(frame)
  • 创建的结果是：
    state  year  pop
0    Ohio  2000  1.5
1    Ohio  2001  1.7
2    Ohio  2002  3.6
3  Nevada  2001  2.4
4  Nevada  2002  2.9

索引与切片

  • 可以对某一列或者某一行直接索引

PIL库

  • image是PIL库中的一个图像的类
  • 使用前需先安装pillow；

Image

  • 读取图片：img.open()
  • 显示图片：show()
    • 直接调用的时候是用计算机上的查看图片的软件显示
    • 调用前调用imshow()即可输出图片
  • mode图像的模式
  • size图像的大小
  • ratate()旋转
  • crop()剪切，参数是一个四元组，前两个是裁剪矩形左上角的坐标，后两个是右下角的坐标
  • resize()缩放，参数是一个二元组，分别是缩放后的长和宽。
  • transpose()镜像，传入Image.FLIP_LEFT_RIGHT时为左右镜像，传入Image.FLIP_TOP_BOTTOM时为上下镜像。
